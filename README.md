## 🔒 Privacy & Enterprise Isolation
Subnet Lab Pro is architected ground-up for high-security, air-gapped corporate data centers. 
* The application runs **100% offline**.
* No analytics trackers, external data collection libraries, or third-party tracking metrics are embedded.
* Your internal IP allocations, subnet masks, and system network layouts never leave your device. 

*Please do not post confidential architectural data, corporate IPs, or sensitive system details within public GitHub issues.*
